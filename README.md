Institutional bilingual website for Wakama's agricultural credit infrastructure platform.

## Stack

Next.js 15, TypeScript strict, Tailwind CSS 4, next-intl, shadcn/ui, lucide-react, framer-motion, react-hook-form, zod.

## Local commands

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run format
```
